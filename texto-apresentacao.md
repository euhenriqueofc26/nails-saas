## 📈 Para Investidores

**O ClubNailsBrasil é um SaaS B2B focado no maior mercado de beleza do mundo: o Brasil.**

### O Mercado

- O Brasil é o **2º maior mercado de beleza do mundo**, atrás apenas dos EUA
- Existem **mais de 1 milhão de nails designers** em atividade no país
- A maioria absoluta ainda gerencia o negócio **manualmente**: caderno, WhatsApp solto, planilha
- Menos de 5% usa software de gestão — **oportunidade gigante de digitalização**

### O Produto

Plataforma SaaS completa que resolve a operação diária da nail designer:

- **Agendamento online** com página pública própria
- **CRM** com histórico e fotos por cliente
- **Gestão financeira** (receita, despesa, lucro)
- **Automação via WhatsApp** (confirmação, lembretes, promoções)
- **Onboarding guiado** — baixíssimo atrito pra começar
- **Plano gratuito funcional** como porta de entrada

### Modelo de Negócio

| Plano | Preço | Margem estimada |
|-------|-------|-----------------|
| Gratuito | R$ 0 | Aquisição |
| Pro | R$ 29,90/mês | ~85% |
| Premium | R$ 59,90/mês | ~85% |

**Ciclo de venda:** Trial grátis → conversão para Pro → upgrade para Premium

### Diferenciais Competitivos

1. **Verticalizado para unhas** — não é um genérico, é feito PARA nails designers
2. **WhatsApp nativo** — a ferramenta que a profissional já usa todos os dias
3. **Onboarding em 3 passos** — 5 minutos e já está usando
4. **Custo operacional baixo** — Next.js + Neon + Vercel = infraestrutura enxuta
5. **Página pública com SEO próprio** — cada nail designer ganha visibilidade online

### Tecnologia

- **Stack moderna:** Next.js 14, TypeScript, TailwindCSS, PostgreSQL (Neon)
- **Hospedagem:** Vercel (escalável globalmente)
- **Storage:** Cloudinary (25GB gratuitos para imagens)
- **Autenticação:** JWT
- **Pagamento:** Simulado (pronto para integrar com Stripe/Asaas)
